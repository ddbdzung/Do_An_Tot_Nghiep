# Installation

- Install [Node.js] (https://nodejs.org/en/download/)
- Install [Git] (https://git-scm.com/downloads)

```
git clone https://github.com/ddbdzung/Do_An_Tot_Nghiep.git
cd Do_An_Tot_Nghiep
cd back-end
npm install -g pnpm
pnpm install
```

# Run

```
pnpm run dev
```

# Stop

```
Ctrl + C
```
